# Engineering Mathematics Dataset

#### Author: Moin Ahmed (moinahmed100@gmail.com)
#### Date Started: Jan 2026

This repository contains files of the dataset intended for a large language model (llm) in the field of advanced enginnering mathematics.

The work is ongoing and any suggestions or contributions are greatly appreciated. Please contact the author at the email: moinahmed100@gmail.com


## Note for Developers
### Recommended Installation

1. Create python virtual environment.
2. Install the following packages into the python virtual environment via pip.
    <code>pip install datasets</code>
